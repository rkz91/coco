# Source 05 — TechCrunch: Cohere Transcribe open-source voice model

Fetched: 2026-05-27
URL: https://techcrunch.com/2026/03/26/cohere-launches-an-open-source-voice-model-specifically-for-transcription/

## Headline facts

- **Model:** Transcribe
- **Release date:** March 26, 2026
- **Size:** 2 billion parameters
- **License:** Open source
- **Hardware target:** Consumer-grade GPU deployment
- **Throughput:** 525 minutes of audio processed per minute
- **Quality:** Average word error rate (WER) of 5.42
- **Languages:** 14 — English, French, German, Italian, Spanish, Portuguese, Greek, Dutch, Polish, Chinese, Japanese, Korean, Vietnamese, Arabic

## Positioning

- Targets both enterprise and self-hosted use cases.
- Plans to integrate Transcribe into the North enterprise agent orchestration platform.
- Offered free through the Cohere API and the Model Vault managed inference platform.

## IPO context

- TechCrunch references Gomez saying the startup may go public "soon."

## Multilingual roadmap

- Current 14-language coverage; gaps noted in Portuguese, German, and Spanish relative to competitors — signals continued focus on multilingual parity.

## Notes for synthesis

- Transcribe is a small, specialized model — embodies the "smaller, specialized > giant generalist" stance Gomez took on 20VC.
- Open-source release continues the Apache 2.0 / Command A+ pattern of using openness as a competitive lever against closed-API rivals.
- Voice is the next frontier for enterprise agents (call centers, dictation, meeting transcription).
