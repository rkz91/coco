3. Gap Analysis

Navigation Gaps (Getting Lost)

GapWhereProblemSeverity
Inbox tab overload InboxPage 8 tabs is decision paralysis. User opens inbox and freezes: All? Urgent? Drafts? Actions? Classify? Health? Suggestions? Auto-handled? GAP
ProjectDetail is a kitchen sink ProjectDetailPage 10 tabs crammed into one page. 1000+ LOC. User wanted to check agents but ends up in the wrong tab. No clear entry point. GAP
No "back to where I was" Global After acting on a todo or reviewing an agent, there's no breadcrumb trail or "return to context". User restarts from Home. FRICTION
Scope context is invisible ScopeContext Clicking a tree node scopes the whole app but there's no persistent visual indicator. User forgets they're scoped. FRICTION
Dead routes in CommandPalette CommandPalette /people and /tasks commands exist but pages don't. Cmd+K search sends you nowhere. DEAD

Action Gaps (Can't Do the Thing)

GapWhereProblemSeverity
No quick action from Home HomePage Home shows what's happening but you can't ACT from there. Every action requires navigating to another page. Should be: see it, do it, done. GAP
Agent spawning is buried AgentsPage To spawn an agent: navigate to agents page → find/create agent → click spawn → fill task. Should be: "do this" from anywhere. GAP
No inline todo completion HomePage / Inbox Can see todos on Home but must navigate to TodosPage to mark done. Should be a checkbox right there. FRICTION
Chat is disconnected ChatPage Chat with Claude exists but is a separate island. Can't easily say "chat about this agent's output" or "chat about this todo". FRICTION
No batch operations TodosPage / InboxPage Can't select 5 todos and mark all done. Can't approve 3 drafts at once. Everything is one-by-one. FRICTION

Feedback Gaps (Don't Know What Happened)

GapWhereProblemSeverity
Agent output is a wall of text LogViewer Agent finishes → user reads a raw log stream. No summary, no "here's what I did". Should show structured output first, raw log on expand. GAP
No success/failure summary AgentsPage Agent completes → card says "completed". But WHAT did it produce? No preview, no summary, no deliverable link. FRICTION
Silent failures Multiple ScopeContext fails silently. Inbox classify fails silently. Chat stream drops silently. No error states. FRICTION
Cost surprise CostsPage No real-time cost indicator while agents run. You find out the spend AFTER. Should show running cost on agent card. FRICTION

4. The Simplified Journey (Target)

Key principle: Home is the hub. Everything flows OUT from Home and BACK to Home. No page-hopping. Slide-over panels for detail. Inline actions everywhere. The Orb is the universal "do something" button.

5. Fix Priority List

#FixWhat ChangesImpact
1 Inline actions on Home cards Add approve/complete/dismiss buttons directly on Home briefing cards. No navigation needed. Eliminates 60% of page-hops
2 Collapse Inbox to 3 tabs Urgent + Drafts + Everything Else. Kill: Classify, Suggestions, Health, Auto-handled, Actions tabs. Kills decision paralysis
3 Agent summary card on completion When agent finishes, show structured summary (what it did, files changed, key output) not raw log. Makes agent output usable
4 Quick spawn from Orb CoCo Orb becomes "do this" button: type task → picks best agent → spawns. No navigate to AgentsPage. Agent spawning in 2 clicks
5 Split ProjectDetailPage Replace 10-tab monster with: Overview page + slide-over panels for each section. Reduces cognitive load
6 Persistent scope indicator Show current scope in top bar: "Viewing: AuditBoard > AB Config" with clear X to unscope. User always knows context
7 Running cost on agent cards Show live token count + estimated USD on the agent card while it runs. No cost surprises
8 Contextual chat Add "Chat about this" button on agent output, todo detail, knowledge content. Opens chat with context pre-loaded. Connects the islands
9 Error states everywhere Add visible error UI for: scope failure, SSE disconnect, classify failure, chat stream drop. Stops silent failures
10 Remove dead routes Remove /people and /tasks from CommandPalette. Remove unused Zustand dep. Clean up. Less confusion
